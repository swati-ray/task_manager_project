# Task Manager API

A RESTful API scaffold for a simple task manager application built with Django REST Framework.

This scaffold includes:
- Django project and apps (`users`, `tasks`)
- JWT authentication (Simple JWT)
- Task model + CRUD endpoints
- Filtering, search, pagination
- API docs via drf-spectacular
- Unit tests
- GitHub Actions CI workflow

See `requirements.txt` for dependencies and follow the README instructions to run locally.
